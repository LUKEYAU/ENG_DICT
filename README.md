英文生字學習工具 - Spelling Master


## 使用


1. **準備題庫**：建立一個 `words.json` 檔案（格式參考下方）。
2. **啟動工具**：開啟 `index.html` 或訪問部署好的 GitHub Pages。
3. **上傳檔案**：點擊上傳區域選擇你的 `.json` 檔案。
4. **開始練習**


## JSON 題庫格式範例

```json
[
  {
    "word": "technology",
    "definition": "科技、技術",
    "etymology": "源自希臘語 'tekhne' (工藝/技能) + 'logia' (研究/學說)。"
  },
  {
    "word": "astronaut",
    "definition": "太空人"
  }
]
```
